Python
Pandas
NumPy
Matplotlib
Seaborn

🔍 Steps Performed

-Data Cleaning
-Handled missing values
-Converted date_added to datetime
-Feature Engineering
-Extracted year_added and month_added
-Split duration into numeric values
-Data Analysis
-Compared Movies vs TV Shows
-Analyzed top countries and genres
-Studied rating distribution
-Visualization
-Bar charts
-Pie charts
-Trend analysis using line plots

**📈 Key Insights
-Movies are more than TV Shows
-Content increased rapidly after 2015
-Drama and Comedy are most popular genres
-USA and India produce most content**-
